## 第二部分：5步转化引擎操作层

> 任何视频脚本，必须严格依次完成这5步心理转化，缺一不可。

| 步骤 | 目标 | 强制公理 | 素材库/操作清单 |
|------|------|----------|-----------------|
| 1. 物理吸引 (0-3秒) | 打破刷视频的惯性 | "3秒视觉/听觉奇观"：钩子即结论，不铺垫不设问 | ① AI生成前后对比图（强烈视觉冲击）② 反向结论断言（"别再学提示词了！"）③ 高冲突对话（"老板太过分了！"）④ 直接展示海量方案刷屏的震撼感 |
| 2. 身份代入 (3-15秒) | 建立"这说的就是我"的共鸣 | "痛点情景镜像"：一句讲完 | ① "你是不是也最怕甲方说'大气'？"② "新手设计师面对提示词一头雾水？"③ "广告店老板接到急单，设计师不够用？" |
| 3. 欲望构建 (15-45秒) | 展示方案，激发强烈渴望 | "三句话演示法"：丢进去→选参数→点生成 | ① 输入：上传现场图/参考图+选行业/输名称② 处理：点击生成③ 输出：展示效果图 |
| 4. 信任强化 (45秒-结尾前) | 消除疑虑，确认专业度 | "演示即信任"：效果图本身就是证明，信任词自然融入演示 | ① "60人设计团队调教的知识库"② "广告老手都在用"③ 演示中配字幕"落地级商用" |
| 5. 行动指令 (结尾) | 明确转化路径 | "低门槛互动" | ① "评论区发现场图，免费出方案"② "点赞收藏，免得要用时找不到"③ "打出你的行业，下期出你的方案" |

## 第四部分：开放式情景公理库

> 永续创意的冲突母题

### 4.1 降维打击类

- **核心冲突公式：** 传统方法 vs 棘手问题 → AI不可思议解决
- **角色库：** 资深设计师、竞品公司
- **场景库：** 提案会、会议室
- **可变参数：** 对手身份、设计类型
- **示例：** "马上提案了，渲染来不及！""不用渲染，我直接改。"30秒出新图。

### 4.2 紧急救援类

- **核心冲突公式：** 极限死线 → 常规手段无解 → AI瞬间救场
- **角色库：** 着急的老板、崩溃的同事
- **场景库：** 临投标前、客户等改稿
- **可变参数：** 死线时间、丢失内容
- **示例：** "完了！源文件损坏，明天开展！""别慌。"JPG复刻，几分钟出图。

### 4.3 识货/不识货类

- **核心冲突公式：** 角色A轻视我方 → AI产出远超预期 → 态度反转
- **角色库：** 传统老板、低预算客户
- **场景库：** 报价谈判、客户比稿
- **可变参数：** 轻视点、反转程度
- **示例：** "就几张图还这么贵？""我当着您面做，看值不值。"1分钟6版方案。客户当场签合同。

### 4.4 新概念颠覆类

- **核心冲突公式：** 旧流程低效 → AI"全案智能体" → 宣告新范式
- **角色库：** 行业观察者、平台方
- **场景库：** 创意策划会
- **可变参数：** 对比流程、整合物料
- **示例：** 传统设计流程割裂。科幻熊猫输入品牌名和行业，自动生成全案，风格统一。

## 第五部分：文体形式创意引擎

> 核心公理：
> - **公理一：** 形式是内容的"放大器"，选哪种形式，取决于你想放大哪种情绪。
> - **公理二：** 形式是无限的"跨界组合"，几乎所有爆款 = 基础形式 × 风格滤镜。

### 5.1 基础原型

| 类型 | 特点 | 字数上限 |
|------|------|----------|
| 口播类 | "你"和"我"的对话感 + 屏幕录播 | 200字 |
| 动画类 | 高视觉冲击的动态图形 + 快节奏文案 | 150字 |
| 剧情类 | 基于冲突母题的影视化拍摄 | 250字 |

### 5.2 风格滤镜库

| 滤镜类别 | 风格滤镜名称 | 特点/叠加方法 | 适配原型 |
|----------|-------------|---------------|----------|
| **影像质感类** | 纪录跟拍 | 手机Vlog，边走边录，真实感 | 口播、剧情 |
| | 监控/偷拍 | 模拟办公室监控视角 | 剧情 |
| | 王家卫式 | 抽帧、文艺独白 | 口播 |
| **节目包装类** | 新闻播报 | 正式着装，标题式开场 | 口播 |
| | 产品发布会 | 极简PPT式动画 | 动画 |
| | 设计庭审 | 模拟法庭辩论 | 剧情 |
| **娱乐跨界类** | 搞笑配音 | 影视剧台词与AI操作同步 | 动画、剧情 |
| | 游戏界面 | 鼠标变准星，功能选武器 | 动画 |
| | 默剧/卓别林式 | 无对白，夸张肢体，滑稽音效 | 剧情 |

### 5.3 新物种形态库

**创意对赌类：**
- **核心：** 以设计为赌注，公开挑战粉丝。
- **结构：** 设立挑战 → 过程记录（Vlog） → 公布结果并@粉丝，形成闭环。
- **目标：** 强互动、涨粉。

**物料漂流类：**
- **核心：** 打造不存在的"公益设计服务"。
- **结构：** 征集需要设计却没预算的街角小店 → 用平台生成改造方案 → 打印并送给店主，记录惊喜。
- **目标：** 强共鸣、提升品牌美誉度。
